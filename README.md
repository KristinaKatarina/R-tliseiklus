# R-tliseiklus
Rüütliseiklus on pygame'i kasutades tehtud arvutimäng. Mäng on tehtud Tartu Ülikooli Programmeerimine I kursuse raames ning autoriteks on Juhan Pauklin ja Kristina Katarina Kaljumäe.
